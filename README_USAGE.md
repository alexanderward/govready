
# Usage

Run the script `parse.py` to convert desired `.csv` files into `.json`


| Variable         | Type      | Value                                                                                                                     |
|------------------|-----------|---------------------------------------------------------------------------------------------------------------------------|
| -i               | String    | Path to input csv file                                                                                                    |
| -o               | String    | Path to output json file                                                                                                  |


Example:
`python parse.py -i data/800-171-source.csv -o output.json`
